EECS 448 Lab03 Description: 

This lab covers the basic usage of HTML, CSS and JavaScript in developing webpages. The lab consists of the following exercises: 

Exercise 1: Password Validation
JavaScript is used to validate a password input. It requires the password to be entered twice and for the password to be at least 8 characters long.
Exercise 2: Image Slideshow
JavaScript is used to create a slideshow that cycles through 5 images by using two input buttons. The slideshow wraps back to the beginning when pressing next at the end of the queue, and vice versa.
Exercise 3: Profile Page
A profile page for Ursula K. Le Guin to demonstrate using CSS to style various element properties, such as font family, text color, background color and more.
Exercise 4: CSS Manipulation
JavaScript is used to dynamically manipulate CSS properties of a paragraph element. Input is read from various text input elements, and a try-catch block is used to validate the entered text.
